# what-to-dou

A simple interactive in-memory CLI Todo application that allows users to manage tasks through a menu-driven interface. This first version provides a basic yet effective way to create, view, update, delete, and mark tasks as complete/incomplete.

## Current Version: 1.0.0

The initial release of what-to-dou introduces core functionality for managing your tasks through an intuitive command-line interface:

- **Task Management**: Create, update, and track your tasks with a simple menu system
- **Interactive Menu**: Easy-to-use menu with options 1-6 for different operations
- **In-Memory Storage**: Fast task management with temporary storage during the session
- **Task Operations**: Add, update, delete, view, and toggle completion status of tasks

This version represents the foundation of our application with essential features to help you stay organized and productive through a command-line interface.
